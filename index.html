CORRECIONES DE PROTOTIPO

PIEZAS1

IMAGEN: Se puede observar una ampliación en la parte de la muñeca del extensor cubital, esto se debe a que en la primera prueba se detctó falta de rigidez. Se debe recordar que el objetivo del extensor es disminuir los movimientos involuntarios de la mano, por lo que se necesita proporcionar rigidez sin lastimar la muñeca.

Código muestra para movimiento de servo

//https://programarfacil.com/tutoriales/fragmentos/servomotor-con-arduino/

// Incluímos la librería para poder controlar el servo

#include

// Declaramos la variable para controlar el servo

Servo servoMotor;

void setup() {

// Iniciamos el monitor serie para mostrar el resultado

Serial.begin(9600);

// Iniciamos el servo para que empiece a trabajar con el pin 9

servoMotor.attach(9); }

void loop() {

// Desplazamos a la posición 0º

servoMotor.write(0);

// Esperamos 1 segundo

delay(1000);

// Desplazamos a la posición 90º

servoMotor.write(90);

// Esperamos 1 segundo

delay(1000);

// Desplazamos a la posición 180º

servoMotor.write(180);

// Esperamos 1 segundo

delay(1000);

}
